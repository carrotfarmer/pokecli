# pokecli

A CLI to find information about Pokemon!

- Built using `React Ink`

![screenshot](https://i.imgur.com/3IV6kc1.png)

## Install

```bash
$ npm install --global @pokecli/pokecli
```

## CLI

```
Usage
  $ pokecli

Options
  --name The name of the pokemon

Examples
  $ pokecli --name=charmander
  Charmander
  ----------
  Metrics: 0.6m, 8.5 kg
  Type: fire

  Stats
  hp: 39
  attack: 52
  defense: 43
  special-attack: 60
  special-defense: 50
  speed: 65
```
